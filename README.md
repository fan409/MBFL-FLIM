MBFL-FLIM

Overview

This project contains code used in experiments on Mutation-Based Fault Localization (MBFL), with a focus on analyzing and mitigating Fault Localization Interference Mutants (FLIMs).

FLIMs are a class of mutants that may negatively affect fault localization effectiveness. This repository includes scripts for FLIM analysis, fault localization evaluation, and result visualization under different mutation strategies (e.g., Major and μBERT).

---

Structure

- src/ : source code for FLIM analysis, fault localization, and plotting
- results/ : selected experimental results

---

Usage

Scripts in src/ can be run directly

Notes

- This repository provides the core implementation used in the experiments.
- Some scripts depend on specific environments and external datasets, and may require additional configuration.
- Due to size limitations, full datasets are not included.
